# SCOVION website

Open `index.html` in a browser to preview the static site. The site defaults to English and remembers a visitor's selection in the browser. Available languages are English, Simplified Chinese, Traditional Chinese, Bahasa Melayu and Thai.

Pages: Home, AI Healthcare (Infervision-led), AI Agriculture, AI Agent, About, and Contact. The healthcare page uses three product visuals derived from the user-supplied KPJ presentation; regulator labels are based on Infervision's official product overview and should be confirmed for the intended local market before publishing.
